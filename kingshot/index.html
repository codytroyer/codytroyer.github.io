Roadmap

The immediate focus is “decision leverage” tools: things that save time and reduce mistakes under resource constraints.

Phase 1 — Foundations

  • Core calculators (resource, upgrades, timing)
  • Consistent data model + versioning
  • Transparent assumptions and formulas

Phase 2 — Optimization

  • Strategy planners and “what-if” comparisons
  • Constraint optimization (budget, time, event windows)
  • Shareable builds/plans

Phase 3 — Community Scale

  • Community-sourced datasets (with validation)
  • Contribution guidelines + issue templates
  • Open methodology and reproducible results

About this project

The goal is practical: build tools that help players make better choices with limited resources. Everything here is designed to be understandable, inspectable, and community-friendly.

Transparency

Formulas and assumptions should be visible and easy to audit.

Local-first

Prefer computations in the browser. Avoid collecting player data.

Iterate quickly

Start small, ship improvements, and validate with player feedback.
